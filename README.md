# Sustainable Bags App

## Overview
The Sustainable Bags App is a web application built with Flask that aims to promote sustainable practices by providing users with information and resources related to eco-friendly bags. The application features user authentication, dashboards for different user roles, and a clean, responsive design.

## Project Structure
```
sustainable-bags-app/
├── app.py                # Main application file for the Flask app
├── models.py             # Data models for the application
├── requirements.txt      # List of dependencies
├── static/               # Static files (CSS, JS, images)
│   ├── css/
│   │   └── style.css     # CSS styles for the application
│   ├── js/
│   │   └── scanner.js    # JavaScript code for client-side functionality
│   └── images/
│       └── logo.png      # Logo image for the application
└── templates/            # HTML templates for the application
    ├── index.html        # Main homepage template
    ├── login.html        # Login page template
    ├── signup.html       # Signup page template
    ├── user_dashboard.html# User dashboard template
    ├── cafe_dashboard.html# Cafe dashboard template
    └── admin_dashboard.html# Admin dashboard template
```

## Installation
1. Clone the repository:
   ```
   git clone https://github.com/yourusername/sustainable-bags-app.git
   cd sustainable-bags-app
   ```

2. Create a virtual environment:
   ```
   python -m venv venv
   ```

3. Activate the virtual environment:
   - On Windows:
     ```
     venv\Scripts\activate
     ```
   - On macOS/Linux:
     ```
     source venv/bin/activate
     ```

4. Install the required packages:
   ```
   pip install -r requirements.txt
   ```

## Usage
To run the application, execute the following command:
```
python app.py
```
The application will be accessible at `http://127.0.0.1:5000`.
